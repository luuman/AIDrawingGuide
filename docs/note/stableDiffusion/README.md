# AIDrawingGuide
这是一个开源的中文AI绘画教程文档及社区，旨在帮助所有对AI绘画感兴趣的学习者在 AI 时代到来的当下更好地学习和利用Stable Diffusion, MidJourney等工具提高自身的工作、绘画效率、更好地享受生活。

本教程包括但不限于一以下内容：
- Stable Diffusion，MidJourney安装和使用教程
- 绘画提示词(prompt)指南
- AI绘画行业发展动态、落地项目分析
- AI绘画实用工具站及经典案例分享
- AI交流群
虽然叫做「ChatGPT」教程，但我们的内容同样适用于「文心一言」等基于 GPT 的 AI 产品。
